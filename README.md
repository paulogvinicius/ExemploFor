# Estruturas de Repetição em Java: `for`

Este repositório contém exemplos e explicações práticas sobre o funcionamento do laço de repetição `for` na linguagem Java. Essa estrutura é essencial para automatizar tarefas repetitivas de forma limpa e eficiente.

---

## 🚀 Como Funciona?

O laço `for` agrupa três comandos essenciais na sua própria declaração, separados por ponto e vírgula `;`:

```java
for (inicialização; condição; incremento/decremento) {
    // Código que será repetido
}
